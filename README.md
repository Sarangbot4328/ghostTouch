# 고스트터치

개인용 Android 터치 매크로 앱입니다. 앱을 실행해 오버레이 권한과 접근성 권한을 켠 뒤, 다른 앱 위에 뜨는 고스트터치 UI에서 설정, 실행, 중지를 조작할 수 있습니다.

## 현재 기능

- 다른 앱 위에 플로팅 UI 표시
- 플로팅 UI 버튼: 설정, 실행, 중지, 끄기
- 실행/중지 상태를 버튼 문구와 색상으로 표시
- 설정 화면에서 넘기기 기능 ON/OFF
- 넘기기 실행 간격을 초 단위 범위로 설정
- 예: 10~30초로 저장하면 10초에서 30초 사이의 랜덤 시점마다 한 번씩 위로 스와이프
- 중지를 누를 때까지 반복

## 휴대폰에서 권한 켜는 순서

1. 고스트터치 앱을 설치하고 실행합니다.
2. `오버레이 권한 열기`를 눌러 다른 앱 위에 표시 권한을 허용합니다.
3. `접근성 설정 열기`를 눌러 `고스트터치 제스처 서비스`를 켭니다.
4. `넘기기 설정`에서 `넘기기 기능 사용`을 체크하고 시간 범위를 저장합니다.
5. `고스트터치 UI 띄우기`를 누릅니다.
6. 원하는 앱으로 이동한 뒤, 떠 있는 고스트터치 UI에서 `실행`을 누릅니다.
7. UI를 없애고 싶으면 `끄기`를 누릅니다.

## GitHub Actions로 APK 빌드

이 저장소를 GitHub에 올리면 `.github/workflows/build-apk.yml` 워크플로가 debug APK를 빌드합니다.

수동 빌드 방법:

1. GitHub 저장소의 `Actions` 탭으로 이동합니다.
2. `Build APK` 워크플로를 선택합니다.
3. `Run workflow`를 누릅니다.
4. 빌드가 끝나면 artifact `ghosttouch-debug-apk`를 다운로드합니다.

APK 위치:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## 참고

이 앱은 Play Store 배포용이 아니라 개인 사용 APK 기준으로 작성되었습니다. 일부 보안 앱, 결제 화면, 은행 앱, 오버레이 차단 화면에서는 Android 정책상 동작이 제한될 수 있습니다.
